# GCN-CIS

## ✈️ Highlights
🔥 A novel segmentation network named GCN-CIS, which is enhanced with a Prior Boundary Attention Module.  
🔥 A Geometrically Constrained Loss function tailored specifically to the unique morphological characteristics of the optic cup and disc in fundus images.  
🔥 GCN-CIS achieves outstanding performance on our self-constructed Glaucoma Screening Dataset named GSD and three other publicly available datasets.  
🔥 A Website based on this network can assist doctors in diagnosis.
